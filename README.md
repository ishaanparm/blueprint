# Blueprint NYC

Blueprint NYC is a simple website that helps New Yorkers find important local help and resources faster.

The goal is to make it easier for people to find support for things like food, housing, benefits, school help, jobs, mental health, and immigrant services without getting overwhelmed by too many complicated websites.

Live site: https://blueprint-nyc.org

## Why I built this

Many helpful NYC resources already exist, but finding the right one can be confusing. Blueprint NYC organizes useful links and information in a cleaner, easier way so people can quickly understand where to start.

This project is especially focused on students, families, and people who may not know which NYC service to use first.

## What the website includes

* Simple categories for different types of help
* NYC-focused resource information
* Clear explanations of what each resource is for
* A clean layout that works on phones and computers
* A visible disclaimer that this is not an official NYC government website

## Important disclaimer

Blueprint NYC is an independent student project. It is not an official NYC government website.

Information can change, so users should always confirm details directly with the official organization or agency before making decisions.
